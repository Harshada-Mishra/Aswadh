Interrupts in 8085 Microprocessor


INTERRUPT: Interrupts are the signals generated by the external devices to request the Microprocessor to perform a task.There are 5 interrupt
signals i.e.TRAP,RST 7.5,RST 6.5,RST 5.5 and INTR. Following are the categories in Interrupt.

1)VECTOR INTERRUPT: In this type of interrupt, the ISR(Interrupt Service Routine) or Interrupt address is known to the processor.e.g.RST 7.5,RST6.5,RST5.5,TRAP.

2) NON-VECTOR INTERRUPT: In tis type of interrupt, the ISR address is not known.Hence, the Interrupt address needs to be sent externally by the device.eg.INTR.

3) MASKABLE INTERRUPT: In this type of interrupt we can disable the interrupt by writing some instructions into the program.e.g. RST 7.5, RST 6.5, RST 5.5.

4)NON-MASKABLE INTERRUPT: In this type of interrupt we can't disable the interrupt by writing some instructions into the program.e.g. TRAP.

5) SOFTWARE INTERRUPT: In this type of interrupt, the programmer has to add the instructions into the program to execute the interrupt. There are 6
Software Interrupts in 8085 e.g. RST 0, RST 1, RST 2, RST 3, RST 4, RST 5, RST 6, RST 7.

6) HARDWARE INTERRUPT: There are 5 interrupt pins in 8085 used as hardware interrupts e.g. TRAP, RST 7.5, RST 6.5, RST 5.5, INTA.
