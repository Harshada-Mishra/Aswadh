FEATURES OF HTML 5:

1) It's a Non Case-sensitive language.
2) It's a Non Procedural language.
3) It's a Scripting language.
4) It is going to be represented with the help of tags [< >].

TAGS:

a. Tags are known as 'Elements'.
b. Each element can have zero or more 'Attributes'.
c.ATTRIBUTE: Characteristics of elements are known as Attributes.
d. Elements may have common and uncommon/Exclusive Attributes.
e. Each Attribute has values.

TWO TYPES OF TAG:


a. EMPTY TAG: Tags which are not required to be closed is known as Empty Tag. E.g. (br),(hr),(frame)etc..
b. CONTAINER TAG: Tags which are required to be closed is known as Container Tag. E.g. (html),(body)etc..
