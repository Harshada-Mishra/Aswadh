BISECTION METHOD:

Q.1] Find root of the Eq. x3-10=0. Using Bisection Method.
soln: Let f(x)= x3-10= 0
f(0)= (0)3-10 = -10
f(1)= (1)3-10 = -9
f(2)= (2)3-10 = -2 < 0
f(3)= (3)3-10 = 17 > 0
Here, f(2) < 0 and f(3) > 0
Root lies between (2,3)
By Bisection Method,
