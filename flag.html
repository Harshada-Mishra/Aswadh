FLAG REGISTER

FLAG REGISTER: A Flag Register defines(shows) the status of Operation performed by Microprocessor. It is a 8-bit Register which has '5 Flip-Flops such as follows:-
a. Carry Flag(CY)
b. Parity Flag(P)
c. Auxiliary Carry Flag(AC)
d. Zero Flag(Z)
e. Sign Flag(S)

a. CARRY FLAG(CY): Carry Flag is the carry generated in the result of any Operation. If carry is 'Present' in the result then the carry flag is Set to '1'.
If carry is 'Absent' in the result then the carry flag is Reset to '0'.

b. PARITY FLAG(P): Parity Flag counts the number of 1 in the result of the Operation except carry.If 'Even number of 1's are present in the result, then
Parity Flag is Set to '1'. If Odd number of 1 is present in the result, then Parity Flag is Reset to '0'.

c. AUXILIARY CARRY FLAG(AC): Auxiliary Flag is the carry generated in between the operation. If carry is 'Present' in between the operation then the AC
Flag is Set to '1'. If Auxiliary carry is 'Absent' in between the operation then then AC Flag is Resetted to '0'.
