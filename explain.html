Pin Description of 8085 Microprocessor

1) Pin no.40(Vcc)& Pin no. 20(Vss): These pins provide Supply Voltage to Microprocessor.

2) Pin no.21-28 (A8-A15): This pins has in all 8 address lines and are used to send Higher Order Address along it.

3) Pin no.12-19 (AD0-AD7): This pins has 8 address lines & 8 data lines which are multiplexed pins where it is used to send Lower Order Address and data along it.
In this pin only data or address can be sent one at a time.Hence, ALE Pin comes where it tells us whether data is sent or adress along pin no.12-19.

4)Pin no.30(ALE): ALE is an abbreviation for Address Latch Enable which is Active high for address. It means when ALE pin is high(1) then (AD0-AD7)
is carrying lower order address along it whereas,when it is low(0)then 12-19 pin is carrying lower order data along it.

5) Pin no.32(RD): This signal indicates that the selected IO/M(External device or memory device)is to be read and is ready for accepting data available on
data trees.This pin is Active low i.e.when the value of this pin is '0' then 'Read' pin is activated.

6) Pin no.31(WR): This signal indicates that the data on the data bus is to be written into selected memory or IO location. Read and Write can be done in IO or
memory device.Hence,next pin arises to let us know whether Read or Write Command is applied on IO device or Memory device.

7)Pin no.34( IO/M ): This signal is used to differentiate between IO and Memory operations i.e.When it is high(1) it indicates IO operation. Whereas, when it
is low(0) then it indicates operation on Memory device.

8) Pin no.39(HOLD): HOLD pin interrupts the microprocessor to inform that external or IO device wants to access its resources.

9) Pin no.35(Ready): Ready pin informs IO/External device that it is ready to give the access to its resources.
