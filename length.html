LENGTH OF THE INSTRUCTION:

1) 1 BYTE: When one memory location is required to store data then it is known as 1 byte instruction. Opcode always takes 1 byte memory. E.g. MOV A,B

2) 2 BYTE: When two memory location is required to store data then it is known as 2 byte instruction. E.g. MVI A,37H

3) 3 BYTE: When three memory location is required to store data then it is known as 3 byte instruction. E.g. LXI B,1234H.

ADDRESSING MODES:

1) DIRECT ADDRESSING MODE:
a) Address appears after Opcode of instuction.
b) The address of operand is specified within instruction.
c) These are generally 3-byte instruction.
d) E.g. LDA D500H
This instruction will load Accumulator with the content of address D500H.
